# ADR-004: TipTap for Document Editing

**Status:** Accepted  
**Date:** 2026-07-14  
**Context:** Need a professional document editor for agreements, proposals, and general documents.  
**Decision:** Use TipTap (ProseMirror).  
**Rationale:** WYSIWYG; extensible; tables, images, comments, variables; version history.  
**Consequences:** Larger bundle; needs custom extensions for variables and AI.  
**Affected Docs:** 02_PRD, 08_SDD
