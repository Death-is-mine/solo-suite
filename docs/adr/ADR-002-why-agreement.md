# ADR-002: Agreement Terminology

**Status:** Accepted  
**Date:** 2026-07-14  
**Context:** Need a term for the legal/business documents exchanged with clients.  
**Decision:** Use "Agreement" instead of "Contract".  
**Rationale:** More approachable; covers proposals, NDAs, SOWs, change requests.  
**Consequences:** All code, UI, and docs use "Agreement".  
**Affected Docs:** 02_PRD, 06_DDD, 18_Glossary
